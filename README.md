# ⏱️ React Countdown Timer

A simple and interactive countdown timer built with React. Users can input a number of seconds, start the timer, and receive an alert when the time runs out. The app includes Start and Restart controls and live countdown display.

## 🚀 Features

- Input custom countdown time in seconds
- Start and stop the timer
- Alert when the countdown finishes
- Clean and minimal UI using Tailwind CSS

## 🛠️ Tech Stack

- React
- Tailwind CSS
- JavaScript (ES6)
